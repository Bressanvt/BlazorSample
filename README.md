# 🚀 BlazorSample

Projeto desenvolvido para praticar o desenvolvimento de aplicações web com **Blazor** e **.NET 10**.

## 🛠️ Tecnologias

- .NET 10
- ASP.NET Core Blazor
- C#
- HTML
- CSS

## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/Bressanvt/BlazorSample.git
```

Entre na pasta do projeto:

```bash
cd BlazorSample
```

Execute a aplicação:

```bash
dotnet run
```

Ou utilize o Hot Reload:

```bash
dotnet watch
```

## 📚 Objetivo

Este projeto foi criado para estudar os principais conceitos do Blazor, como componentes, roteamento, layouts e criação de interfaces utilizando C#.
